# reports/ — 夏夜将终官网整理报告索引

本目录为供人工确认的报告与中间数据，**未改动任何网站代码或 config.json**。确认后可据此更新官网。

## 需要你人工确认的报告（按重要性）

1. `演出时间线.md` — 基于"待处理素材"+微信公众号爬取整理的详细演出时间线，含歌单推断与"待确认"清单。**重点核对**。
2. `内容贡献指南.md` — 给非技术成员和 AI 看的"如何加图片/新闻/作品/演出"操作手册。
3. `乐队核心信息.md` — 乐队档案（成员/年表/作品/友链），供 AI 直接上手。
4. `代码结构与贡献流程问题.md` — 当前实现的可维护性/贡献体验问题清单与优先级建议（只读分析，未改代码）。

## 中间/原始数据（可留作存档）

- `_crawl_raw.json` / `_crawl_raw.txt` — 微信推送爬取结构化与可读结果。
- `_articles/01..09_*.txt` — 9 篇推送的分篇可读版（标题/账号/发布时间/正文/图片 URL）。
- `_crawl.py` / `_dump.py` / `_download.py` — 抓取脚本（可复跑）。

## 说明

- 小红书素材按指示跳过（SPA 无法抓取）。
- 海报内节目单/歌单因运行环境不支持图片识别，已标注"待人工确认"并列出海报 URL。
